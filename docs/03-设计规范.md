# UI 设计规范

## 配色方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主背景色（侧边栏） | `#1B2838` | 深蓝，左侧导航栏背景 |
| 主背景色（内容区） | `#F5F7FA` | 浅灰白，右侧内容区背景 |
| 卡片/面板背景 | `#FFFFFF` | 纯白，图表区、表格区背景 |
| 主色调（按钮、强调） | `#2E86DE` | 蓝色，查询/计算/保存按钮 |
| 主色调 Hover | `#1E6FBF` | 深蓝，按钮悬停态 |
| 成功/正收益 | `#27AE60` | 绿色，正收益率文字 |
| 危险/负收益 | `#E74C3C` | 红色，负收益率文字、删除按钮 |
| 文字主色 | `#2C3E50` | 深灰蓝，主要文字 |
| 文字次要色 | `#7F8C8D` | 灰色，辅助文字 |
| 边框色 | `#E0E4E8` | 浅灰，分隔线、表格边框 |

## 字体规范

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 大标题 | 22px | Bold | 页面标题 |
| 小标题 | 16px | Bold | 区块标题 |
| 正文 | 14px | Normal | 表格内容、输入框 |
| 小字 | 12px | Normal | 提示文字、辅助信息 |

字体：Microsoft YaHei（微软雅黑，Windows 自带）

## 布局规范

- 窗口默认大小：1200 × 750 像素
- 最小窗口大小：900 × 600 像素
- 左侧导航栏宽度：200 像素
- 内容区内边距：24 像素
- 卡片圆角：10 像素
- 按钮圆角：8 像素

## 组件规范

### 导航按钮
- 高度：44 像素
- 字体颜色：白色（`#FFFFFF`）
- 默认态：透明背景
- 选中态：浅蓝背景（`#2E86DE`），加左侧 4px 白色指示条
- Hover 态：半透明背景（`rgba(255,255,255,0.1)`）

### 操作按钮
- 查询/计算/保存按钮：蓝色背景 `#2E86DE`，白色文字
- 删除按钮：红色背景 `#E74C3C`，白色文字
- 高度：36 像素，内边距左右 24 像素

### 输入框
- 高度：36 像素
- 边框：1px 实线 `#E0E4E8`
- 聚焦态：蓝色边框 `#2E86DE`
- 圆角：8 像素

### 表格
- 表头：浅蓝背景 `#EBF0F7`，加粗文字
- 数据行：白色背景，Hover 态 `#F5F7FA`
- 正收益绿色 `#27AE60`，负收益红色 `#E74C3C`

### 图表
- Matplotlib 嵌入，中文支持（微软雅黑）
- 收益率曲线：蓝色 `#2E86DE`
- 基准线（0%）：灰色虚线
- 买入点标记：红色倒三角
- 回本点标记：绿色圆形
- 图表背景白色，网格线浅灰
