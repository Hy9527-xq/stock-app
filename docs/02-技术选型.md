# 技术选型说明

## 技术栈

| 层级 | 技术 | 版本要求 | 用途 |
|------|------|----------|------|
| 编程语言 | Python | 3.11+ | 主语言 |
| GUI 框架 | CustomTkinter | 最新 | 现代化桌面 UI，蓝白主题 |
| 数据来源 | akshare | 最新 | 免费 A 股历史 K 线数据 |
| 图表绘制 | Matplotlib | 最新 | 嵌入式图表（收益率曲线、价格走势） |
| 数据处理 | pandas | 最新 | 时间序列处理、收益率计算 |
| 本地存储 | SQLite | Python 内置 | 保存用户查询记录 |
| 打包分发 | PyInstaller | 最新 | 打包为单个 .exe |

## 选型理由

1. **Python**：金融数据生态最丰富，akshare/pandas/matplotlib 组合成熟稳定
2. **CustomTkinter**：比 tkinter 原生控件更美观，比 PyQt 更轻量，适合中小型桌面应用
3. **akshare**：完全免费开源，无需注册 API Key，覆盖 A 股全量历史数据
4. **SQLite**：Python 内置，零配置，单文件存储，备份方便
5. **PyInstaller**：将 Python 程序打包为独立 exe，用户无需安装 Python

## 不选的技术及原因
- **Electron**：包体太大（~200MB），对于简单的股票查询工具过于臃肿
- **PyQt/PySide**：功能强大但学习曲线陡峭，GPL 协议需要关注
- **在线 API（tushare 等）**：需要注册、积分、token，对小白用户不友好
- **Web 应用（Flask + 浏览器）**：需要启动服务，体验不如原生桌面程序
