# 实施步骤

## 执行原则
- 每一步独立可测试，完成一步验证一步
- 不在一步中修改过多文件
- 每个步骤完成后在开发日志中记录

## 步骤概览

### 第一步：环境搭建
- [ ] 确认 Python 版本（需要 3.11+）
- [ ] 创建虚拟环境
- [ ] 安装核心依赖：`customtkinter`, `akshare`, `matplotlib`, `pandas`
- [ ] 验证依赖导入正常

### 第二步：数据获取层 `data/fetcher.py`
- [ ] 实现 `search_stock(keyword)` — 股票名称/代码模糊搜索
- [ ] 实现 `get_stock_data(code, start_date, end_date)` — 获取历史 K 线
- [ ] 错误处理：网络异常、代码不存在、日期无效
- [ ] 单独测试：运行 fetcher.py 验证数据能正确获取

### 第三步：计算逻辑 `utils/calculator.py`
- [ ] 实现 `calc_return_rate(df)` — 累计收益率序列
- [ ] 实现 `calc_daily_prices(df)` — 每日开盘/收盘价提取
- [ ] 实现 `find_breakeven_date(df, buy_date, buy_price)` — 回本日期计算
- [ ] 单独测试：用模拟数据验证计算逻辑

### 第四步：本地存储层 `data/storage.py`
- [ ] 创建 SQLite 数据库和表结构
- [ ] 实现 `save_record()` — 保存查询记录
- [ ] 实现 `get_all_records()` / `get_records_by_year()` / `get_records_by_month()`
- [ ] 实现 `delete_record()`
- [ ] 单独测试：增删查操作

### 第五步：UI 主框架 `app.py`
- [ ] 创建主窗口（标题、尺寸、蓝白主题）
- [ ] 左侧导航栏（三个按钮：收益率查询、回本计算、我的记录）
- [ ] 右侧内容区（Frame 容器，切换显示不同页面）
- [ ] 页面切换逻辑
- [ ] 启动验证：主窗口正常显示，导航切换正常

### 第六步：功能一页面 `ui/page_return.py`
- [ ] 股票搜索框 + 日期输入 + 查询按钮
- [ ] Matplotlib 嵌入式收益率曲线图
- [ ] 可滚动数据表格（日期、开盘价、收盘价、涨跌幅）
- [ ] 保存按钮 + 保存成功提示
- [ ] 功能测试：输入股票查询，验证图表和表格

### 第七步：功能二页面 `ui/page_recovery.py`
- [ ] 股票搜索框 + 日期 + 买入价输入 + 计算按钮
- [ ] 结果展示区（回本日期、等待天数等）
- [ ] Matplotlib 价格走势图（标记买入点和回本点）
- [ ] "未回本"状态处理
- [ ] 功能测试：验证回本计算正确性

### 第八步：功能三页面 `ui/page_records.py`
- [ ] 汇总切换控件（全部/按年度/按月度）
- [ ] 树形列表展示（年度 → 月份 → 记录）
- [ ] 点击记录 → 跳转到功能一并回显
- [ ] 删除按钮 + 确认对话框
- [ ] 功能测试：保存、查看、删除全流程

### 第九步：整合与打包
- [ ] 编写 `main.py` 入口文件
- [ ] PyInstaller 打包配置
- [ ] 生成 .exe 文件
- [ ] 在无 Python 环境测试
