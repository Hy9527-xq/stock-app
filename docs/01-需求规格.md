# 需求规格说明书

## 项目概述
一款运行在 Windows 上的 A 股（沪深）分析软件，面向非技术用户，操作简单直观。

## 核心功能

### 功能一：收益率查询
- **输入**：股票代码或名称（如"600519"或"贵州茅台"）、起始日期、结束日期
- **输出**：
  - 该时间段内的累计收益率曲线图
  - 每日开盘价、收盘价、涨跌幅数据表格
- **操作**：支持「保存此次查询」到本地记录

### 功能二：回本计算器
- **输入**：股票代码或名称、买入日期、买入价格
- **逻辑**：从买入日期开始往后逐日比较收盘价，找到第一个收盘价 ≥ 买入价的交易日
- **输出**：
  - 回本日期、等待天数、回本当日的收盘价
  - 买入当日收盘价与亏损幅度
  - 如果至今未回本，显示"尚未回本"及当前亏损幅度
  - 价格走势图（标注买入点和回本点）

### 功能三：我的记录（保存/删除/汇总）
- **保存**：功能一查询完成后，可保存查询条件与收益率摘要
- **归类规则**：以查询的**起始日期**为准，归入「年度 → 月份」
  - 例：查询 2026-01-05 ~ 2026-02-25 → 归入「2026年 → 1月」
  - 例：查询 2025-12-20 ~ 2026-01-10 → 归入「2025年 → 12月」
- **查看**：按 年度 → 月份 两层树形结构展示
- **回显**：点击记录可跳转到功能一，回显当时的查询条件并拉取最新数据
- **删除**：每条记录旁有删除按钮，确认后删除

## UI 要求
- 简洁、直观，适合非技术用户
- 主色调：蓝白配色
- 布局：左侧导航栏（深蓝）+ 右侧内容区（白色）
- 软件形式：桌面窗口程序（双击 exe 运行）

## 边界情况处理
- 股票代码/名称不存在 → 提示用户
- 查询日期超出数据范围 → 提示可用范围
- 网络请求失败 → 提示重试
- 保存重复记录 → 允许（同一个股不同时间段可能不同）
- 回本日期不存在（从未回本）→ 显示"尚未回本"
